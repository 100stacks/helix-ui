--- title: also: components/drawer: Drawer --- {% extends 'element.njk' %} {% import '_utils.njk' as utils %} {% block page_header %}

The {{page.title}} is a container for a drawer dialog, anchored to the right-side of the screen. It is used to build a Drawer component.

{% endblock %} {% block content %}
Permitted Parents
any
Permitted Children
In this order:
  1. {{ utils.element('header') }}
  2. {{ utils.customElement('hx-alert') }} (optional)
  3. {{ utils.customElement('hx-div') }}
  4. {{ utils.element('footer') }}

  5. Fixed position elements with slot="fixed" attribute. (optional)

    • {{ utils.customElement('hx-menu') }}
    • {{ utils.customElement('hx-popover') }}
    • {{ utils.customElement('hx-search-assistance') }}
    • {{ utils.customElement('hx-tooltip') }}

    Elements will fail to position without slot="fixed".

    Fixed position elements must be immediate children of {{page.title}}.

Events
Any of the following:
  • close (cancelable)
  • open (cancelable)
{% endblock %} {% block attributes %}
open

Opens the drawer.

Does not emit events on change.

{% endblock %} {% block properties %}
open {Boolean [false]}

Determines if the drawer is open.

Emits open event when set to true.

Emits close event when set to false.

{% endblock %}
