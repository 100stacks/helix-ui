The custom {{page.title}} element provides functionality to navigate between one or more tab panels.

Permitted Parents any Permitted Children one of each:
  • an <hx-tablist> element
  • an <hx-tabcontent> element
 Events Any of the following:
  • tabchange when the current tab changes

Methods

selectNext()
Selects the next tab in the tabset. If the last tab is currently active, selects the first tab.
selectPrevious()
Selects the previous tab in the tabset. If the first tab is currently active, selects the last tab.
{% endblock %} {% block attributes %}
current-tab {Integer=0}

Zero-based index of the currently active tab + tabpanel.
