Attributes

aria-controls
Automatically set by hx-tabset when attached to the DOM.
aria-selected
Automatically set as the current attribute/property changes.
current
Automatically applied by hx-tabset to identify the current Tab in a Tablist.
id
Used by hx-tabset to wire up a Tab to its Tabpanel using the aria-controls and aria-labeled by attributes. If you do not define an id attribute, one will be automatically generated.
role
Automatically set, if not explicitly defined in markup.
