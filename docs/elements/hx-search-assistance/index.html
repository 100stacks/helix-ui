The custom {{page.title}} element contains a list of choices for the user, such as recent or suggested searches depending on their context.

Permitted Parents
any
Permitted Children
flow content
Events
Any of the following:
  • close
  • open
  • scroll (Emitted from document as a CustomEvent instance.)
{% endblock %} {% block attributes %}
open
Opens the search assistance
position {String="bottom-start"}
Positions the search assistance
relative-to (optional)
ID of an element that the search assistance is positioned relative to.
{% endblock %} {% block properties %}
controlElement (HTMLElement) read-only

Returns the HTML element with the aria-controls attribute that matches the <hx-search> id.

open {Boolean [false]}

Manipulates the open attribute. Opens the search assistance.

position {String [bottom-start]}

Manipulates the position attribute.

relativeElement (HTMLElement) read-only

Returns the HTML element used as a reference to calculate the position of the visible search assistance. If the relative-to attribute is set, this will be the element with ID matching the relativeTo property value. Otherwise, this will be the controlElement.

relativeTo {String}

Manipulates the relative-to attribute. ID of an element that the search assistance is positioned relative to.
