The custom {{page.title}} element describes the states of a downloadable asset.

Permitted Parents any Permitted Children flow content Events Any of the following:
  • cancel - User clicks "X" when loading or invalid.
  • delete - User clicks "X" to delete the file.

Methods

dismiss()
Simulates a mouse click on the dismiss button (i.e., clicking the "X")
{% endblock %} {% block attributes %}
name {String} (optional)
Sets the text of the name. If absent or blank, the name will not be displayed.
href {String} (optional)
Sets the path of the href. If absent, file name will not be a clickable link. Avoid setting this attribute until the file exists on the server.
icon {Enum<String>} (optional)
Changes the displayed icon to one of the available icons. If absent, blank, or unrecognized, the icon will not be displayed.
loading {Boolean [false]} (optional)
When present, indicates file upload is in progress.
invalid {Boolean [false]} (optional)
When present, marks element invalid.
{% endblock %} {% block properties %}
name {String}
Manipulates the name attribute.
href {String}
Manipulates the href attribute.
icon {String}
Manipulates the icon attribute.
loading {Boolean}
Manipulates the loading attribute.
valid {Boolean}
Manipulates the invalid attribute.
{% endblock %}
