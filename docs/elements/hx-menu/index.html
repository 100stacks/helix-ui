--- title: minver: 0.2.0 also: components/menus: Menus elements/hx-menuitem: --- {% extends 'element.njk' %} {% block content %}

The custom {{page.title}} element defines a list of choices to the user, such as a list of actions or functions.

Permitted Parents
any
Permitted Children
either of the following:
  • zero or more <hx-menuitem> elements
  • zero or more <section> elements
Events
Any of the following:
  • close
  • open
  • reposition
{% endblock %} {% block attributes %}
open (optional)
Opens the menu
position (optional)
Positions the menu
relative-to (optional)
ID of an element that the menu is positioned relative to. If not set, the menu will be positioned relative to the element with the aria-controls attribute matching the menu ID.
{% endblock %} {% block properties %}
controlElement {HTMLElement} read-only

Returns the HTML element with the aria-controls attribute that matches the <hx-menu> ID.

open {Boolean [false]}

Manipulates the open attribute.

position {String [bottom-start]}

Manipulates the position attribute.

relativeElement {HTMLElement} read-only

Returns the HTML element used as a reference to calculate the position of the open menu. If the relative-to attribute is set, this will be the element with ID matching the relativeTo property value. Otherwise, this will be the controlElement.

relativeTo {String}

Manipulates the relative-to attribute.

{% endblock %}
