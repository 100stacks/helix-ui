Layouts are not automatic. Apply appropriate CSS class to document body. Padding is not automatic. Add Box styling to apply padding.

Two application layouts are available.

  • Vertical application layout
  • Horizontal application layout

Both layouts are optimized for use with single-page applications (SPAs) and they have the same basic HTML structure.

Vertical Layout

You can copy/paste the source code from the vertical layout template to get a head start.

Use a Vertical layout if your application content needs to grow along the y-axis (pushing the footer down past the bottom of the viewport).

  • Apply the .hxVertical class to the document body (<body class="hxVertical">).

Horizontal Layout

You can copy/paste the source code from the horizontal layout template to get a head start.

Use a Horizontal layout if your application content needs to grow across the x-axis (keeping the footer visible at all times).

  • Apply the .hxHorizontal class to the document body (<body class="hxHorizontal">).

App Area

The App area serves as the mounting point for your SPA.

It should contain the Stage as its only child.

Stage

Do not use more than one side rail in your application.

The Stage contains three potential children.

  1. An optional product navigation
  2. Main content
  3. An optional siderail

Product Navigation

The left-hand production navigation should contain navigation links to pages within your application.

  • Use a <nav> element to provide the best support for accessible user agents.
  • Give the element an id of nav to position it correctly in the application layout.
  • Give the element a class of hxNav to apply visual styles.
  • The product navigation is most commonly implemented using a vertical navigation component.

Main Content

Content area behaves like a non-wrapping Grid row when using a Horizontal layout.
  • For accessibility reasons, use a <main> element.
  • Add the role="main" attribute to provide the best support for accessible user agents.
  • Give the element an id of content.
