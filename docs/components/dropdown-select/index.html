Validation

A {{page.title}} is designed to take advantage of standardized HTML form validation capabilities. Error styles will be applied as long as the <select> element matches the :invalid CSS pseudo-class.

There are two ways to modify validity of the <select> element.

  1. add or remove the required attribute
  2. calling its setCustomValidity() method
