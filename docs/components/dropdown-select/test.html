HXSelect - Display Order

We only care about display order, so none of the folowing tests pass accessibility (no select[id] or label[for]).

Because of IE's grid implementation, we cannot auto arrange extra elements. Only label, select, and hx-select elements are supported as direct children of .hxSelect.

  • If present, the label should appear before the control.
  • Extra elements should not be visible.
