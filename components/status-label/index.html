Emphasis Gray

The .-hxEmphasisGray CSS class can be added to status labels that require more attention than the default style.

Emphasis Gray 
<hx-status class="-hxEmphasisGray">
  Emphasis Gray
</hx-status>

Emphasis Purple

The .-hxEmphasisPurple CSS class can be added to status labels that require more attention than the gray emphasis.

Emphasis Purple 
<hx-status class="-hxEmphasisPurple">
  Emphasis Purple
</hx-status>

Subdued

The .-hxSubdued class can be used on status labels that require less or no attention from the user.

Subdued 
<hx-status class="-hxSubdued">
  Subdued
</hx-status>

Filled Labels

The .-hxFill class can be used on status labels to apply filled backgrounds. This class can be used in conjunction with any of the previous status label modifier classes above.

Emphasis Gray
Emphasis Purple
Subdued

  Emphasis Purple
</hx-status><br/>
<hx-status class="-hxFill -hxSubdued">
