Single Button

The .hxBtn classes are designed to be used with the <button> element. However, you can use them on <a> and <input> elements.

Only "button", "submit", and "reset" inputs are supported.
Anchor Link 
<button class="{{loneClasses}}">Button</button>
<a class="{{loneClasses}}">Anchor</a>
<a href="#" class="{{loneClasses}}">Link</a>
<input type="button" class="{{loneClasses}}" value="Input" />
<input type="submit" class="{{loneClasses}}" value="Submit" />
<input type="reset" class="{{loneClasses}}" value="Reset" />

Button Group

<div class="{{ groupClasses }}">
  <button class="hxBtn">First</button>
  <button class="hxBtn">Second</button>
  <button class="hxBtn">
    <hx-icon type="angle-down"></hx-icon>
  </button>
</div>
