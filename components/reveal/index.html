A reveal is a behavior-only web component that implements almost no styling.

Default Reveal

Closed by default

Click me to show content

You can't see me.

 
  <hx-reveal>
    <span slot="summary">Click me to show content</span>
    <p>You can't see me.</p>
  </hx-reveal>

Open Reveal

Add open attribute to set the reveal in an open state

Click me to hide content!

You should see me.

 
  <hx-reveal open>
    <span slot="summary">Click me to hide content!</span>
    <p>You should see me.</p>
  </hx-reveal>

Attributes

open
Opens the reveal

Functions

The following functions are available on the element.

close()
Closes the reveal
open()
Opens the reveal
toggle()
Inverts the open state
