<hx-toast> NEW Requires the latest version of HelixUI

The custom <hx-toast> element provides a user with information for action outside of the user's current context.

Permitted Parents any Permitted Children phrasing content Events Any of the following:
  • dismiss (cancelable) - User clicks "X" to dismiss the toast.
  • submit - User clicks CTA

Methods

dismiss()
Dismisses the toast.

Attributes

Attributes allow you to configure an element in a declarative manner via markup.

cta {String} (optional)
Set the text of the CTA button. If absent or blank, the CTA button will not be displayed.
type {Enum<String>} (optional)
Identifies the type of toast. Valid values are as follows:
  • info
  • error
  • success

Properties

Properties allow you to configure an element in an imperative manner via JavaScript.

cta{String}
Manipulates the cta attribute.
type{String}
Manipulates the type attribute.
