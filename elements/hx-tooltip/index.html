<hx-tooltip>

The custom <hx-tooltip element displays supplemental information related to an element on the page.

Position:

Appearance

position

Left position: left position: left-top position: left-bottom

Attributes

Attributes allow you to configure an element in a declarative manner via markup.

id (String) Required

Links element to target data-element attr placed on the target element.

NOTE: Currently this will only find the first element with data-tooltip attr on the page.

Properties

Properties allow you to configure an element in an imperative manner via JavaScript.

open (Boolean [false])
Determines whether or not tooltip is shown
position (String [top])
Determines where tooltip is positioned
trigger-event (String [hover])
Determines what fires the tooltip possible values:
  • click
  • hover
