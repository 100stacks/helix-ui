<hx-tablist>

The custom <hx-tablist> element arranges tabs within a tabset.

Permitted Parents an <hx-tabset> element Permitted Children zero or more <hx-tab> elements

Attributes

Attributes allow you to configure an element in a declarative manner via markup.

role
Automatically set to tablist, if not explicitly defined in markup.

Properties

Properties allow you to configure an element in an imperative manner via JavaScript.

This element does not have any configurable properties.
