<hx-popover>

The custom <hx-popover> element defines a section of floating content to appear nearby other elements.

Permitted Parents any Permitted Children flow content

Popover Header

<button data-popover="myPopover" class="hxBtn">Toggle</button>

<hx-popover id="myPopover" position="bottom-right">
  <header slot="header">
    <h4 class="hxContainerTitle">Popover Header</h4>
  </header>
  <p>This is my popover body</p>
  <footer slot="footer">
    <button class="hxBtn hxBtn--primary">Submit</button>
    <button class="hxBtn">Cancel</button>
  </footer>
</hx-popover>

Attributes

Attributes allow you to configure an element in a declarative manner via markup.

open
Opens the popover

Properties

Properties allow you to configure an element in an imperative manner via JavaScript.

open (Boolean [false])
Determines if the popover is open.
position (String [bottom-right])
Determines the position of the popover.
