<hx-panelhead> v0.2.0+ HelixUI v0.2.0 or later required

The custom hx-panelhead element aligns content to the top of a panel.

Permitted Parents
an <hx-panel> element
Permitted Children
flow content
Events
none

Attributes

Attributes allow you to configure an element in a declarative manner via markup.
