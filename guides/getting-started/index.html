Getting Started with HelixUI

This guide is intented to help developers initialize a Helix application.

Step 1: Install HelixUI via npm or Yarn

HelixUI is registered as a package on npm. You can install the latest version with the npm CLI command: 

npm install --save-dev helix-ui

Alternatively, you can use the Yarn CLI command: 

yarn add helix-ui

Both approaches install HelixUI in the node_modules/ directory.

Step 2: Choose a Layout

Building a Helix application begins with choosing a layout. HelixUI provides two supported layouts from which you can choose.

  1. Vertical Layout
    • Lengthy content pushes the footer below the fold.
  2. Horizontal Layout
    • The footer is always visible.

Step 3: Add Content

Add content and interactivity to your application using components. Components contain interactive demos and information for various interaction patterns.

  1. Choose a component from the navigation on the left.
  2. Check the "See Also" section on the component page for more information.
